# Bajaj Finserv Backend

## Run Locally

Clone the project

```bash
  git clone https://github.com/kmabhinai/Bajaj-Finserv-Backend
```

Go to the project directory

```bash
  cd web-scrapper
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
